了解数据相关的业务详情

  • SalePrice - 以美元为单位的物业销售价格 这是你试图预测的目标变量
  • MSSubClass - 建筑类
  • MSZoning - 一般的分区划分
  • LotFrontage - 街道的线性脚与物业相连
  • LotArea - 以平方英尺为单位的土地面积
  • Street - 进入道路的类型
  • Alley - 胡同入口的类型
  • LotShape - 属性的一般形状
  • LandContour - 酒店的平坦度
  • Utilities - 可用的实用程序类型
  • LotConfig - 批量配置

转换非数字特征（以Street参数为例）

  • 对Street数据探究

使用scikit-learn来创建一个训练集和一个保留集

scikit-learn的train_test_split()函数返回四个对象：

  • X_train 是我们用于培训的功能的子集。
  • X_test 是将成为我们的“保留”集合的子集 - 我们将用来测试模型。
  • y_train 是SalePrice相应的目标变量X_train。
  • y_test 是SalePrice相应的目标变量X_test。

参数值

  • X表示预测变量数据集，并且y是目标变量。
  • random_state=42，这提供了可重复的结果，
  • train_test_split将随机分配数据。该test_size参数告诉函数在test分区中应该有多少数据。在这个例子中，大约33％的数据专用于保留集。

评估性能并可视化结果

评估函数

Kaggle将使用均方根误差（RMSE）评估我们的提交。

R^2 is: 0.888350909967

  • model.score()方法默认返回r平方值
  • r平方值是衡量数据与拟合回归线有多接近的度量。它取值在0和1,1之间，这意味着目标中的所有变化都由数据解释。一般来说，较高的r平方值意味着更合适。

预测模型

RMSE is: 0.0178253278631

  • model.predict()方法将返回给定一组预测变量的预测列表。model.predict()在拟合模型后使用。
  • mean_squared_error函数需要两个数组并计算rmse
  • RMSE：测量我们的预测值和实际值之间的距离

可视化
